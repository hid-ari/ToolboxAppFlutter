# 🧰 ToolBox

Aplicación móvil que reúne varias utilidades en una sola app. Permite consultar información usando diferentes APIs públicas como predicción de género por nombre, estimación de edad, universidades por país, clima en República Dominicana, información de Pokémon y noticias desde WordPress.

# 📦 Concepto de la App

La aplicación funciona como una **caja de herramientas digital**, ya que reúne múltiples utilidades en una sola aplicación.
